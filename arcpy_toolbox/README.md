# arcpy_toolbox

This folder contains the **ArcGIS Python Toolbox** implementation of the
Polygon-to-Centerline tool.

## Contents

| File | Description |
|---|---|
| `Polygon_to_Centerline.pyt` | ArcGIS Python Toolbox (`.pyt`) — add this to ArcGIS Pro or ArcMap |

## Requirements

- ArcGIS Pro 2.x+ **or** ArcGIS Desktop 10.x
- **Standard** or **Advanced** license (required for `Densify`, `Feature Vertices To Points`, `Create Thiessen Polygons`, `Polygon To Line`, and `Smooth Line`)

## How to use

1. In **ArcGIS Pro** (Catalog pane) or **ArcCatalog**, right-click the folder
   and choose **Add Toolbox**, then select `Polygon_to_Centerline.pyt`.
2. Run the **Polygon to Centerline** tool.

See the full algorithm and parameter documentation in the root
[README.md](../README.md).
